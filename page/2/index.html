Spring Boot 是简化 Spring 应用开发的一个框架块，它整合了 Spring 的整个技术栈，提供了 J2EE 开发一站式解决方案。

RestTemplate 类是为调用REST服务而设计的，它的主要方法与 HTTP 协议方法紧密相连， 比如它有 headForHeaders()、getForObject()、postForObject()、put()和delete()等方法。
