Jenkins 插件安装失败解决办法

发表于 2020-04-06 更新于 2022-03-25 分类于 DevOps 阅读次数： Valine：

前言

如果还没有安装 Jenkins 的小伙伴，可参考另一篇博文： CentOS7 安装 Jenkins

Jenkins 的官方镜像源非常慢，国内基本不可用，经过一番研究整理，有3种安装插件解决办法

  1. 在 Jenkins 官网下载想要的插件，再导入到 Jenkins 中。(较为繁琐)
  2. 更改 Jenkins 的镜像源。(可用，但安装插件还是不够快)
  3. 同第二点类似同样要配置镜像源，不过要先安装中文社区的简体中文插件。(推荐，安装插件较快)

下面详细说明这 3 种解决办法

解决办法

导入 Jenkins 插件

首先进入官网插件界面， https://plugins.jenkins.io/

输入插件名称搜索，比如搜索 localization, 如果想更换简体中文语言，红框内3个插件是必须的

下载插件

选择版本，注意，如果想用第三种解决办法，这里要选择 1.0.10 以上

下载完进行插件管理页面

选择 Advanced(高级)一栏，导入该插件即可

Jenkins 更换镜像源

配置国内镜像源，先到镜像源站点查看可用的镜像源：http://mirrors.jenkins-ci.org/status.html

这里使用清华大学开源软件镜像站的镜像源：

在 Jenkins 中配置该镜像源

尝试下载 Jenkins 中文社区简体中文插件, 选择 Avaliable(可选插件)一栏，Filter输入框输入：Localization

点击左下侧 Install without restart 进入安装页面，如果失败了，就再试多几次。

使用中文社区镜像源

这种解决办法是推荐的，可彻底解决插件安装慢问题，不过要先安装中文插件，可以先通过前面两种方法安装该插件，安装完后是这样的

切换为中文

重要的来了，Jenkins 页面中拉到底部，点击 ==Jenkins 中文社区==

出现如下页面，先点击使用，再点击设置地址

更换镜像源为：

点击提交后，点击 check now(立即获取)

接下来，面对疾风吧，安装一个 Pipeline 插件，很快就安装完了，没有报失败。
